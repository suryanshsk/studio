# **App Name**: Avanish.AI

## Core Features:

- Hero Section: Visually appealing landing section with name, role, tagline, and clear call-to-action buttons, social media links.
- About Me Section: Professional introduction highlighting passions and skills in AI, web development, and content creation.
- Skills Grid: Categorized grid showcasing proficiency in Frontend, Backend, AI/ML/Data, and Other Tools.
- Projects Showcase: Display a portfolio of featured projects with attractive cards, tech stack tags, and 'View Details' links.
- AI Voice Assistant Project (Cognova): Showcase an AI-powered voice assistant project with details on its capabilities and technologies.
- Contact Section: Interactive contact form with social media links.
- AI Code Assistant Tool: AI code generation assistant that uses existing GitHub repository contents as tool to generate code suggestions.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) for a futuristic and professional feel.
- Background color: Dark gray (#212121) for a modern, dark theme.
- Accent color: Electric blue (#03A9F4) for highlighting interactive elements.
- Headline font: 'Space Grotesk', a modern sans-serif for a tech-forward aesthetic. Body font: 'Inter', a versatile sans-serif for readability.
- Code font: 'Source Code Pro' for clean and readable code snippets.
- Use minimalistic, glowing line icons to match the futuristic theme.
- Single-page scrolling layout with sticky navbar and smooth scrolling navigation. Fully responsive design for all devices.
- Subtle animations on hover for buttons and project cards. Smooth scroll reveal animations for sections.
